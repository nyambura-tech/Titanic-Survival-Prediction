# Titanic Survival Prediction
# Overview
This project uses machine learning to predict whether a passenger survived the Titanic disaster based on features such as age, gender, passenger class and fare. The workflow includes data cleaning, exploratory data analysis (EDA), model training and evaluation.

# Features
Data preprocessing and cleaning
Exploratory Data Analysis (EDA)
Feature engineering
Machine learning model training
Model performance evaluation

# Libraries Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Dataset

# Visualization 
Heatmap
Barchart
Boxplot
Histogram

The project uses the Titanic dataset, which contains passenger information and survival outcomes.

# Results
The trained model predicts passenger survival and helps identify the key factors that influenced survival during the Titanic disaster.
